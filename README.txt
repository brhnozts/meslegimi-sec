MESLEGIMISEC FINAL PAKET

Klasör yapısı:
- index.html
- meslek_universite/
- meslek_lise/

Kullanım:
1) Zip'i çıkar.
2) Klasörü VS Code ile aç.
3) Kök klasördeki index.html dosyasını Live Server ile çalıştır.
4) Ana sayfadan Üniversite Yolu veya Lise Sonrası Meslek alanına gir.

Notlar:
- Her alt klasörde ilgili JSON dosyaları aynı klasörde tutulmuştur.
- Üniversite sistemi: meslekler.json, meslek_universite.json, meslek_lise_resmi.json
- Lise sistemi: meslekler_lise.json, meslek_lise_meb.json
